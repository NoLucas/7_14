# ☕ Cafe Moment

카페 주문 웹 서비스입니다. 고객은 메뉴를 둘러보고 장바구니에 담아 주문하며, 라떼아트가 가능한 메뉴는 잔마다 원하는 모양을 선택할 수 있습니다. 관리자는 별도 대시보드에서 메뉴·주문·홈페이지 콘텐츠를 관리합니다.

**배포 주소**: https://nolucas.github.io/7_14/

## ✨ 주요 기능

### 고객
- 메인 페이지 — 인기 메뉴, 오늘의 추천, 라떼아트 갤러리(관리자가 등록한 영상 자동재생), 바리스타 소개
- 메뉴 목록/상세 — 카테고리 필터, 검색, 라떼아트 모양별 탐색(하트/로제타/튤립/곰돌이) 및 미리보기 영상
- 장바구니 — 수량 조절, 담긴 메뉴 관리
- 주문하기 — 결제방법 선택(카드/현금, 매장 결제) + 라떼아트 가능 메뉴는 **잔마다 개별로** 모양 선택 → 주문 완료 시 고유 주문코드 발급
- 주문 내역 — 주문 상태 확인, 라떼아트 요청/영상 확인
- 로그인/회원가입 — Supabase Auth 기반 계정 로그인
- 마이페이지

### 관리자
- 대시보드 — 전체 주문/매출/품절 현황 요약
- 사이드 네비게이션 — 호버(PC)/탭(터치)/키보드 포커스로 열리는 슬라이드 패널
- 메뉴 관리 — 메뉴 CRUD, 품절 처리, 라떼아트 메뉴별 미리보기 영상 여러 개 업로드/삭제
- 주문 관리 — 주문 목록/상세 조회, 상태 변경, 라떼아트 요청 영상 업로드
- 홈페이지 관리 — 메인 페이지 라떼아트 갤러리 영상 4개 슬롯 업로드/교체/삭제
- 관리자 전용 접근 제어(RLS + 로그인 가드)

## 🛠️ 기술 스택

| 구분 | 기술 |
|------|------|
| 프런트엔드 | HTML / CSS / Vanilla JavaScript (프레임워크 없음) |
| 백엔드/DB | [Supabase](https://supabase.com) — Postgres, Auth, Storage. `@supabase/supabase-js`로 클라이언트에서 직접 접근 |
| 인증/권한 | Supabase Auth + `profiles.role` + RLS(`is_admin()`) |
| 로컬 저장 | `localStorage` — 장바구니 등 브라우저 세션 상태만 |
| 폰트 | Pretendard |
| 개발 서버 | [`serve`](https://www.npmjs.com/package/serve) |
| 배포 | GitHub Actions → GitHub Pages |

## 🚀 시작하기

```bash
npm install
npm start
```

`http://localhost:3113` 에서 확인할 수 있습니다.
